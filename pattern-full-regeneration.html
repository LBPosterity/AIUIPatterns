Full Result Regeneration

Overview

AI usually provides a single result per prompt, this pattern simply re submits and regenerates the entire previous prompt result.

User Archetype

This pattern is more applicable for the context of the content rather than the users ability. It's primarily a question of the users expectation of the systems flexibility. More technically proficient users ay want more granular control.

Considerations

This pattern is optimised for succinct prompt with shorter results. Consider if the results your users are generating are more subjective & if full regeneration could assist ideation.

Next consider if regenerating the results require edits to the initial prompt or if users could benefit from contextually relevant pre canned prompts such as "make more succinct or make less technical".
