Dynamic Suggestions

Overview

This pattern analysis the user current incomplete input to then use AI itself to provide suggestion for prompt completions. Exactly like the grammar suggestions we're more familiar with, these suggestions also direct users on quality prompt authoring.

User Archetype

As a passive pattern, many user types can benefit from quality suggestions. Being that you can accept or ignore these suggestions novice users can benefit from quality examples, while advanced users can save time when authoring.

Considerations

Tuning these suggestions would be key to ensuring their relevance. While the suggestions are useful, if a user does not want them, they can become annoying quickly. Consider how many suggestions should be provided & does this pattern benefit your particular workflow & UI.
