Prompt Templates

Overview

Whilst AI generation is fast, a well crafted prompt can take time & experimentation. Boost users effectiveness by allowing them to save & access a library of pre written prompts.

User Archetype

For novice users a library of pre written prompts can be highly effective, helping avoid the arresting barrier of a blank page & providing immmediately effective results for them to build confidence in using AI.

When it comes to advanced users alowing them to save & re use their own prompts leverages their previous hard work & makes experimentation easier.

Considerations

Saving templates can be a simple feature, simply saving & recalling a prompt definition. Hoever, consider how this feature can best support your users workflow. Are they able to update an existing template? Is there a dedicated template editor interface or is the same core prompt UI used for template editing as well.

Providing a library of pre written prompts can be effective but only if they align with your users goals. Do you know common prompt types for your users & can you effectively pre author an effective prompt for them?
