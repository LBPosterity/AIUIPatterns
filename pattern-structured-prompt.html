Structured Prompt

Overview

For situations where a prompt has identifiable key sections, providing them as separate labeled input areas guides the user to provide the required prompt composition.

The separate sections are them combined to a single prompt submission.

User Archetype

A great pattern for novice users, this highly scaffolded approach provides strong guidance being that all prompt sections are required before submission.

Equally though this pattern can be used for users who frequently author prompts in tightly defined contexts as this pattern provides the prompt structure, reducing this aspect of cognitive load, allowing them to solely focus on authoring content.

Considerations

For this pattern to be effective the overall prompt structure must be well defined & required for every submission. Potentially some sections could be marked optional, however this should be limited as this pattern optimised users focus on content over prompt structure.
