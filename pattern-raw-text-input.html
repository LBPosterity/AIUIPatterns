Raw Text Input

Overview

The simplest & most straightforward input, the raw text provided by the user is used to prompt the AI directly.

This is where AI UI have begun, but we have quickly moved away from, although we may never fully remove the humble text input.

User Archetype

Essentially for all user types, the humble textarea box is the most familiar & accessible way to allow prompt authoring. Whilst it gives a user the most control over the prompt, it's most effective user type would require some base knowledge of how to write an effective prompt.

Considerations

They key consideration here is "Does my user need direct control or could they be helped with some abstraction of the UI input controls".

Like the rest of the patterns provide, do they need assistance with prompting from an educational stand point, feedback on prompt quality or would they benefit from highly abstracted UI inputs to craft complex prompts.
